## 📁 Projects

### ✅ Task 1 — Iris Dataset Exploration & Visualization
**Goal:** Understand how to load and explore a dataset using Python.  
**Dataset:** Iris Dataset

### ✅ Task 2 — Credit Risk Prediction
**Goal:** Predict whether a loan applicant will default on a loan.  
**Dataset:** Loan Prediction Dataset (Kaggle)  
**What I did:**
- Cleaned missing values and prepared the data
- Visualized key features like loan amount and income
- Trained a Logistic Regression / Decision Tree classifier
- Evaluated using accuracy and confusion matrix

**Skills:** Data cleaning, Binary classification, Model evaluation

---

### ✅ Task 3 — Customer Churn Prediction
**Goal:** Identify which bank customers are likely to leave.  
**Dataset:** Churn Modelling Dataset  
**What I did:**
- Preprocessed the data and encoded categorical features
- Trained a classification model
- Analyzed feature importance to understand churn drivers

**Skills:** Encoding, Supervised classification, Feature importance

---

### ✅ Task 4 — Medical Insurance Cost Prediction
**Goal:** Estimate insurance charges based on personal details.  
**Dataset:** Medical Cost Personal Dataset  
**What I did:**
- Trained a Linear Regression model to predict charges
- Visualized how BMI, age, and smoking affect costs
- Evaluated performance using MAE and RMSE

**Skills:** Regression modeling, Correlation analysis, Error metrics

---

### ✅ Task 5 — Personal Loan Acceptance Prediction
**Goal:** Predict which customers are likely to accept a loan offer.  
**Dataset:** Bank Marketing Dataset (UCI)  
**What I did:**
- Explored features like age, job, and marital status
- Trained a classifier to predict loan acceptance
- Extracted business insights from results

**Skills:** EDA, Classification, Business insights
